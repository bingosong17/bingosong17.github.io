照片预处理

一、照片管理

1、软件：

Bridge

2、命令

  • 全屏放大查看：空格

  • 比较查看：选中多图使用按钮 Ctrl + B / Command +B

  • 评星：在放大查看的情况下 Ctrl + 数字 / Command + 数字。是用来过滤照片。总共5星，通常按照3星来评。6，7，8，9是标颜色。

  • 照片打包成组：将一系列相似照片打包成组，使其不占空间，比如延时照片。将系列照片都选中后，Ctrl + G / Command + G 。

    注意：拍照必须使用RAW格式

二、整体调整

1、过程–技术指标

  • 准确的色彩还原
  • 正确的黑白场
  • 足够的清晰度
  • 适当的饱和度
  • 丰富合理的层次

2、命令

  • 在Bridge中用Camera Raw打开图片：Ctrl + R / Command + R。双击打开的图片是进入到PS中

3、操作（所有操作在Camera Raw中进行）

（1）使用白平衡工具，找18%中性灰。可以在拍摄中使用白平衡灰板。在实际拍摄中，先拍摄一张白平衡灰板照片，后期以其为基准，进行白平衡调节。或者在图中找衣服边，类似18%中性灰的地方取色。

（2）配置的同步：选择多张后，用Camera Raw打开，第一张白平衡矫正后，可以进行同步设置到其他照片。具体操作是选中要同步的其他照片，右键后，点击“同步设置”。或者在全选中的情况下，修第一张照片。

（3）黑白场调节。调节“黑色”和“白色”滑条。注意，结合ALT / Option 去使用。例如：按住Option滑动“黑色”时，界面黑色部分为死黑，即该处没有细节，纯黑色。同理，如果是滑动“白色”，图片出现白色区域则是惨白，没有一点信息。

（4）清晰度。动了清晰度，就影响曝光。调高清晰度后，曝光增加，可以进行曝光减少，修正。锐化通常是在出片前的操作。锐化是关于轮廓线。通常锐化只需要对局部处理，可以使用ALT / Option 加蒙版的滑动使用。出现白色处为锐化生效区域

（5）适当饱和度。饱和度：所有的饱和度都被提纯。自然饱和度：把那些比较隐现的颜色提纯。鼓励使用自然饱和度。

（6）丰富合理的层次。让直方图的信息丰富起来

4、照片输出

（1）Camera Raw中，右键后，点击-》存储图像

注意

色彩空间问题：相机端使用Adobe RGB，该色域比sRGB大很多，sRGB只用于互联网

（2）批处理时，全部选中后，点击存储图像。或者在PS软件中，点击：文件->脚本->图像处理器
